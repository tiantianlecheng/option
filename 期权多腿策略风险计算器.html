计算公式说明
* 本计算器计算的是到期日损益（时间价值 = 0）。实际持仓中的当前盈亏会因剩余时间价值而不同。
* 风控审核报告自动检测：流动性、零盘口、深实值门槛、盈亏比分级、时间价值分析。
* 盈亏比分级标准：RR < 1:0.5 拒绝 / RR < 1:1 警告 / RR >= 1:1 通过。
* 深实值门槛（现货替代）：买腿纯时间价值 <= 标的现价 x 1% 才合格。
麦克米伦时间价值法 — 选卖购/卖沽行权价

理论基础：市场对短期波动的预期，完全反映在"平值期权的纯时间价值"之中。
天花板（看多/牛差用）= 现价 + 2 x 平值Call纯时间价值 → 卖购行权价不应超过此值
地板（看空/熊差用）= 现价 - 2 x 平值Put纯时间价值 → 卖沽行权价不应低于此值
