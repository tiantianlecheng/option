期权风险计算学习手册

从裸买到铁鹰 · 五步法算清任何策略的最大亏损

第一部分：通用五步法

不管你做什么策略，风险计算永远是这五步：

步骤1: 列出所有腿 (买/卖, Call/Put, 行权价, 权利金)
步骤2: 算净成本 = 买入付出 - 卖出收入 (正数=你花钱, 负数=你收钱)
步骤3: 找"最惨场景"——标的价格到哪个位置你亏最多?
步骤4: 在最惨价格下, 算每腿的到期内在价值
步骤5: 最大亏损 = 净成本 + Σ(每腿到期盈亏)
两个核心公式 (到期日时间价值=0)
Call 内在价值 = max(标的价格 - 行权价, 0)
Put 内在价值 = max(行权价 - 标的价格, 0)

买入腿到期盈亏 = +内在价值 - 权利金 (你花了钱买, 到期收回内在价值)
卖出腿到期盈亏 = -内在价值 + 权利金 (你收了钱卖, 到期要付出内在价值)

第二部分：单腿策略

策略1: 裸买 Call (Long Call)

你的实盘案例: LC2609P168000 是裸买 Put, 逻辑完全一样, 换方向即可。

案例: 买入 SR609C5300, 权利金 65.68, 10吨/手
步骤1 一条腿: 买入 Call, 行权价 5300, 权利金 65.68
步骤2 净成本 = 65.68 (你花钱买了)
步骤3 最惨场景: SR 到期 ≤ 5300, Call 内在价值 = 0, 你一分钱都收不回来
步骤4 在 SR=5300 时: Call内在 = max(5300-5300,0) = 0
步骤5 最大亏损 = 65.68/吨 (全部权利金归零)
总亏损 = 65.68 × 10吨 × 1手 = 656.8 元/手
练习1: 裸买 Put

买入 LC2609P168000, 权利金 11,005, 合约单位 1吨/手, 6张。
问: 最大亏损是多少? 在什么标的价格下发生?

步骤1: 买入 Put, 行权价 168,000, 权利金 11,005
步骤2: 净成本 = 11,005
步骤3: 最惨场景 = LC 到期 ≥ 168,000, Put 内在价值 = 0
步骤4: 在 LC=168,000 时, Put内在 = max(168000-168000,0) = 0
步骤5: 最大亏损 = 11,005/吨 × 1吨 × 6张 = 66,030 元
这就是你之前那笔裸买沽的最大风险——全部权利金归零。

策略2: 裸卖 Call (Short Call) — 无限风险

案例: 卖出 SR609C5400, 收权利金 38.83
步骤1 一条腿: 卖出 Call, 行权价 5400, 权利金 38.83
步骤2 净收入 = 38.83 (你收钱了, 净成本 = -38.83)
步骤3 最惨场景: SR 涨到无穷大 → Call内在 = ∞
步骤4 标的每涨1点, 你亏1点 (delta接近1)
步骤5 最大亏损 = 无限
这就是为什么不能裸卖 Call —— 标的可以无限涨, 你的亏损没有上限。

策略3: 裸卖 Put (Short Put) — 有限但很大

案例: 卖出 SR609P5200, 收权利金 20
步骤1 一条腿: 卖出 Put, 行权价 5200, 权利金 20
步骤2 净收入 = 20 (净成本 = -20)
步骤3 最惨场景: SR 跌到 0, Put内在 = 5200-0 = 5200
步骤4 在 SR=0 时, 你要赔 5200/吨
步骤5 最大亏损 = 5200 - 20 = 5180/吨
总亏损 = 5180 × 10 × 1 = 51,800 元/手

注意: 裸卖Put的最大亏损 = 行权价 - 权利金。不是无限, 但可以非常大。

第三部分：垂直价差 (你做的主要策略)

策略4: 牛市看涨价差 (Bull Call Spread)

你的实盘案例: LH2609 C12400买 / C13000卖, SR609 C5300买 / C5400卖, SA609 C1100买 / C1160卖

案例: SR609 买C5300(65.68) / 卖C5400(38.83), 各6张, 10吨/手
步骤1 腿1: 买入 Call 5300, 付出 65.68
腿2: 卖出 Call 5400, 收入 38.83
步骤2 净成本 = 65.68 - 38.83 = 26.85/吨
步骤3 最惨场景: SR ≤ 5300 时, 两个Call都是虚值, 内在价值都=0
步骤4 在 SR=5300 时:
买C5300内在 = max(5300-5300, 0) = 0
卖C5400内在 = max(5300-5400, 0) = 0
两腿到期价值都=0
步骤5 最大亏损 = 净成本 26.85/吨 (两腿都归零, 亏全部净权利金)
总亏损 = 26.85 × 10 × 6 = 1,611 元

牛差的关键: 卖购不是白卖的——它收了38.83, 降低了你的净成本。如果没有卖购, 你裸买5300C要花65.68; 加了卖购只花26.85。亏损上限从65.68降到了26.85。

练习2: 算你自己的 LH2609 牛差

买入 LH2609C12400, 权利金 415; 卖出 LH2609C13000, 权利金 215.5; 各5张, 16吨/手。
问: 最大亏损是多少? 最大盈利是多少? 盈亏平衡点在哪?

最大亏损:
净成本 = 415 - 215.5 = 199.5/吨
最惨场景: LH ≤ 12400, 两腿都虚值归零
最大亏损 = 199.5 × 16 × 5 = 15,960 元

最大盈利:
最佳场景: LH ≥ 13000, 差宽 = 13000-12400 = 600
组合到期价值 = 600/吨
最大盈利 = (600 - 199.5) × 16 × 5 = 32,040 元

盈亏平衡:
= 买购行权价 + 净成本 = 12400 + 199.5 = 12,599.5

策略5: 牛市看跌价差 (Bull Put Spread) — 收权利金版

卖出高行权价Put + 买入低行权价Put。看多但用卖来表达。

案例: 卖出 SR609P5240(收20) / 买入 SR609P5180(付8), 各6张
步骤1 腿1: 卖出 Put 5240, 收入 20
腿2: 买入 Put 5180, 付出 8
步骤2 净收入 = 20 - 8 = 12/吨 (净成本 = -12, 你收钱了)
步骤3 最惨场景: SR ≤ 5180, 两个Put都是实值
卖Put5240 亏: 5240-SR
买Put5180 赚: 5180-SR
净亏: (5240-SR) - (5180-SR) = 5240-5180 = 60 (差宽, 固定不变)
步骤4 在 SR=5180 时:
卖Put5240内在 = 5240-5180 = 60 (你赔60)
买Put5180内在 = 5180-5180 = 0 (没用上)
总到期支出 = 60
步骤5 最大亏损 = 差宽 - 净收入 = 60 - 12 = 48/吨
总亏损 = 48 × 10 × 6 = 2,880 元

最大盈利 = 净收入 = 12 × 10 × 6 = 720 元 (SR≥5240时两腿归零, 白赚权利金)
盈亏平衡 = 卖Put行权价 - 净收入 = 5240 - 12 = 5228

规律: 收权利金的价差, 最大亏损 = 差宽 - 净收入; 最大盈利 = 净收入。跟花权利金的价差正好相反。

练习3: 验证规律

上面的牛差(花权利金版): 差宽=100, 净成本=26.85
最大亏损 = 净成本 = 26.85 ✓
最大盈利 = 差宽 - 净成本 = 100 - 26.85 = 73.15 ✓

上面的牛差(收权利金版): 差宽=60, 净收入=12
最大亏损 = 差宽 - 净收入 = 48 ✓
最大盈利 = 净收入 = 12 ✓

两种版本的盈亏平衡都是同一个公式: 盈亏平衡 = 低行权价 + 净成本 (花权利金版) 或 高行权价 - 净收入 (收权利金版), 算出来结果一样。

策略6: 熊市看跌价差 (Bear Put Spread)

你的实盘案例: LC2609 P168000买 / P156000卖 (裸买沽转熊差)

案例: 买入 LC2609P168000(付11005) / 卖出 LC2609P156000(收4720), 各6张, 1吨/手
步骤1 腿1: 买入 Put 168000, 付出 11005
腿2: 卖出 Put 156000, 收入 4720
步骤2 净成本 = 11005 - 4720 = 6285/吨
步骤3 最惨场景: LC ≥ 168000, 两个Put都虚值归零
步骤4 在 LC=168000 时, 两腿内在都=0
步骤5 最大亏损 = 净成本 6285/吨
总亏损 = 6285 × 1 × 6 = 37,710 元

最大盈利 = (168000-156000) - 6285 = 12000-6285 = 5715/吨
总盈利 = 5715 × 1 × 6 = 34,290 元
盈亏平衡 = 168000 - 6285 = 161,715

跟牛差完全对称: 净成本就是最大亏损, 差宽减净成本就是最大盈利。只是方向反了。

策略7: 熊市看涨价差 (Bear Call Spread) — 收权利金版

案例: 买入 SR609C5400(付38) / 卖出 SR609C5300(收65), 各6张
步骤2 净收入 = 65 - 38 = 27/吨
步骤3 最惨场景: SR ≥ 5400, 差宽=100全部赔出
步骤5 最大亏损 = 差宽 - 净收入 = 100 - 27 = 73/吨 → 73 × 10 × 6 = 4,380元
最大盈利 = 净收入 = 27/吨 → 27 × 10 × 6 = 1,620元
盈亏平衡 = 5300 + 27 = 5327

注意: 这正好是牛差5300/5400的镜像。同一个行权价, 买/卖方向反过来, 盈亏就镜像翻转。

第五部分：多腿策略 (进阶)

策略8: 铁鹰 (Iron Condor) — 四腿

结构: 卖P_b + 买P_a (下方保护) + 卖C_c + 买C_d (上方保护)

假设 SR609=5270, 做铁鹰:

腿操作行权价权利金
1卖出 Put5200收入 12
2买入 Put5150付出 5
3卖出 Call5350收入 10
4买入 Call5400付出 4
步骤2 净收入 = (12+10) - (5+4) = 13/吨
步骤3 最惨场景有两个方向:
向下最惨: SR ≤ 5150, 下方Put价差全部赔出 = 5200-5150 = 50
向上最惨: SR ≥ 5400, 上方Call价差全部赔出 = 5400-5350 = 50
两边差宽都是50, 所以两个方向最大亏损一样
步骤5 最大亏损 = 差宽 - 净收入 = 50 - 13 = 37/吨
总亏损 = 37 × 10 × 1 = 370 元/手
最大盈利 = 净收入 = 13 × 10 = 130 元/手

铁鹰本质是两个收权利金价差拼在一起, 上下各一个。最大亏损取两边差宽的较大值减净收入。

策略9: 跨式 (Straddle)

买入跨式: 买C + 买P (同行权价)

假设 SR609=5270, 买入 SR609C5300(65) + SR609P5300(60)

步骤2 净成本 = 65 + 60 = 125/吨
步骤3 最惨场景: SR 到期 = 5300, 两腿都正好ATM, 内在都=0
步骤5 最大亏损 = 净成本 125/吨 (标的正好停在行权价不动)
最大盈利 = 无限 (标的大涨或大跌都赚)
两个盈亏平衡: 5300±125 = 5175 和 5425

第六部分：综合练习

练习4: 你自己的 50ETF 牛差

买入 50ETF购7月2900, 成本 0.1484; 卖出 50ETF购7月3000, 收入 0.0705; 各150张, 合约单位10000。
请计算: 最大亏损、最大盈利、盈亏平衡、风险回报比。

净成本 = 0.1484 - 0.0705 = 0.0779/份

最大亏损 = 净成本 = 0.0779 × 10000 × 150 = 116,850 元
(50ETF ≤ 2.900 时两腿归零)

最大盈利 = 差宽 - 净成本 = (3.000-2.900) - 0.0779 = 0.0221 × 10000 × 150 = 33,150 元
(50ETF ≥ 3.000 时差宽全部实现)

盈亏平衡 = 2.900 + 0.0779 = 2.9779

风险回报比 = 116,850 : 33,150 = 1 : 0.28
(这个赔率很差——冒着11.7万的风险只赚3.3万, 成本占差宽的78%)
练习5: 新策略 — 卖沽差

假设你想在 SR609=5270 做一个卖沽差看多:
卖出 SR609P5240, 收权利金 20; 买入 SR609P5180, 付权利金 8; 各6张, 10吨/手。
请完整计算所有风险参数。

步骤1-2: 净收入 = 20 - 8 = 12/吨 (收权利金)

步骤3: 最惨场景 = SR ≤ 5180
卖P5240 赔 5240-SR, 买P5180 赚 5180-SR
净赔 = (5240-SR)-(5180-SR) = 60 (固定差宽)

最大亏损 = 差宽 - 净收入 = 60 - 12 = 48/吨
总亏损 = 48 × 10 × 6 = 2,880 元

最大盈利 = 净收入 = 12 × 10 × 6 = 720 元
(SR ≥ 5240 时两腿归零, 白赚权利金)

盈亏平衡 = 5240 - 12 = 5228

风险回报比 = 2,880 : 720 = 1 : 0.25
(赔率不好, 但胜率高——SR只要不跌破5228就赚, 距现价5270有42点缓冲)
练习6: 挑战题 — 混合策略

假设你做了一个"领口策略"(Collar):
持有 SR609 期货多单(5270开仓) + 买入 SR609P5200(权利金15) + 卖出 SR609C5400(权利金25)
1张期货(10吨) + 各1张期权
请计算: 最大亏损、最大盈利、盈亏平衡。

这是三腿策略, 但逻辑清晰:

腿1 期货多单: 盈亏 = SR - 5270 (涨赚跌亏)
腿2 买Put 5200: 到期价值 = max(5200-SR, 0) - 15 (花钱买的保护)
腿3 卖Call 5400: 到期价值 = 25 - max(SR-5400, 0) (收钱封顶)

净期权现金流 = 25(收) - 15(付) = +10 (净收10)

最惨场景: SR ≤ 5200
期货亏 = SR - 5270
Put赚 = 5200 - SR
Call归零
总盈亏 = (SR-5270) + (5200-SR) + 10 = -5270+5200+10 = -60/吨
最大亏损 = 60 × 10 = 600 元
(不管SR跌到多少, 亏固定60——Put保护了下跌风险)

最佳场景: SR ≥ 5400
期货赚 = SR - 5270
Put归零
Call亏 = -(SR - 5400)
总盈亏 = (SR-5270) + 0 + 25 - (SR-5400) = -5270+5400+25 = 155/吨
最大盈利 = 155 × 10 = 1,550 元
(不管SR涨到多少, 赚固定155——Call封了上涨空间)

盈亏平衡 = 5270 - 10(净权利金收入) = 5260
(净收了10块权利金, 所以盈亏平衡比期货开仓价低10点)

规律: 领口策略本质是"用放弃上涨空间换取下跌保护", 盈亏都被锁定, 像一个隐形的牛差。

总结：你必须记住的几条规律

策略类型最大亏损公式最大盈利公式
花权利金的价差
(牛差花版/熊差花版)		= 净成本
(标的没到目标, 两腿归零)		= 差宽 - 净成本
(标的到了, 差宽全部实现)
收权利金的价差
(牛差收版/熊差收版/铁鹰)		= 差宽 - 净收入
(标的穿过了保护行权价)		= 净收入
(标的在区间内, 两腿归零)
裸买(任何)= 全部权利金
(标的没到你想要的方向)		看方向, 可能很大也可能有限
裸卖Call= 无限= 权利金
裸卖Put= 行权价 - 权利金
(标的跌到0)		= 权利金
