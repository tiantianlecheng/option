# 期权多腿策略风险计算器与学习手册

## 文件说明

- `期权多腿策略风险计算器.html` - 交互式期权组合风险计算工具
  - 支持任意腿数、不同张数
  - ETF期权和商品期权通用（12种合约乘数预设+自定义）
  - 自动计算：净成本、最大亏损、最大盈利、盈亏平衡点、风险回报比
  - 当前市场盈亏（浮盈浮亏）实时计算
  - 到期损益图（红盈绿亏，中国习惯）
  - 5个实盘示例 + 7种策略模板

- `期权风险计算学习手册.html` - 期权风险计算教学手册
  - 五步法通用框架
  - 9种策略详细计算过程
  - 6道练习题（含答案）
  - 速查表

## 使用方法

直接在浏览器中打开 HTML 文件即可使用，无需安装任何软件或依赖。

也可以通过 GitHub Pages 在线访问（永久免费）。
